Business Cases - Solar Powered Irrigation Scheduling & Energy Generation Tracking

1. Solar Powered Irrigation Scheduling: Using Structures and Priority Queues

Description: Solar-powered irrigation scheduling aims to optimize the use of solar energy in irrigation systems. This system ensures that water is only provided to crops when necessary, based on factors such as the solar energy availability, weather forecasts, and the crop's water requirements. The goal is to use solar energy efficiently and reduce water wastage.

C++ Code Example:

#include <iostream>
#include <queue>
#include <vector>
#include <string>

using namespace std;

// Structure to store irrigation details
struct IrrigationTask {
    string cropName;
    int scheduledTime; // Time in hours when irrigation will occur
    int energyRequired; // Energy required for irrigation (in kWh)

    bool operator<(const IrrigationTask& other) const {
        return scheduledTime > other.scheduledTime; // Min-heap based on scheduledTime
    }
};

int main() {
    priority_queue<IrrigationTask> irrigationQueue;

    // Adding irrigation tasks
    irrigationQueue.push({"Wheat", 10, 5});
    irrigationQueue.push({"Rice", 8, 3});
    irrigationQueue.push({"Corn", 12, 4});

    // Display the irrigation schedule
    cout << "Irrigation Schedule (Solar Powered):\n";
    while (!irrigationQueue.empty()) {
        IrrigationTask task = irrigationQueue.top();
        irrigationQueue.pop();
        cout << "Crop: " << task.cropName << ", Time: " << task.scheduledTime << " hours, Energy Required: "
             << task.energyRequired << " kWh\n";
    }

    return 0;
}
Example Output: 
Irrigation Schedule (Solar Powered):
Crop: Rice, Time: 8 hours, Energy Required: 3 kWh
Crop: Wheat, Time: 10 hours, Energy Required: 5 kWh
Crop: Corn, Time: 12 hours, Energy Required: 4 kWh

2. Energy Generation Tracking: Using Linked List

Description: Energy generation tracking refers to monitoring the energy generated by different renewable sources, such as solar panels and wind turbines, over time. The system can help track energy production, identify trends, and make adjustments to optimize energy use. This program maintains a list of energy generation data and outputs the total energy generated by each source.

C++ Code Example:

#include <iostream>
#include <string>

using namespace std;

// Node structure to store energy generation details
struct EnergyGeneration {
    string sourceName;
    double energyProduced; // Energy produced by the source (in kWh)
    EnergyGeneration* next;

    EnergyGeneration(string name, double energy) : sourceName(name), energyProduced(energy), next(nullptr) {}
};

class EnergyTracker {
private:
    EnergyGeneration* head;

public:
    EnergyTracker() : head(nullptr) {}

    // Add a new energy generation record
    void addGenerationRecord(const string& source, double energy) {
        EnergyGeneration* newRecord = new EnergyGeneration(source, energy);
        newRecord->next = head;
        head = newRecord;
    }

    // Display the energy generation records
    void displayEnergyRecords() {
        EnergyGeneration* current = head;
        double totalEnergy = 0;

        cout << "Energy Generation Tracking:\n";
        while (current != nullptr) {
            cout << "Source: " << current->sourceName << ", Energy Produced: " << current->energyProduced << " kWh\n";
            totalEnergy += current->energyProduced;
            current = current->next;
        }

        cout << "Total Energy Generated: " << totalEnergy << " kWh\n";
    }

    // Destructor to free memory
    ~EnergyTracker() {
        while (head != nullptr) {
            EnergyGeneration* temp = head;
            head = head->next;
            delete temp;
        }
    }
};

int main() {
    EnergyTracker tracker;

    // Adding energy generation data
    tracker.addGenerationRecord("Solar Panel", 250.5);
    tracker.addGenerationRecord("Wind Turbine", 180.2);
    tracker.addGenerationRecord("Hydroelectric", 120.3);

    // Display energy generation records
    tracker.displayEnergyRecords();

    return 0;
}
Example Output: 
Energy Generation Tracking:
Source: Hydroelectric, Energy Produced: 120.3 kWh
Source: Wind Turbine, Energy Produced: 180.2 kWh
Source: Solar Panel, Energy Produced: 250.5 kWh
Total Energy Generated: 551 kWh

Summary of the C++ Code Examples:

  • Solar Powered Irrigation Scheduling uses a priority queue to schedule irrigation tasks for crops based on solar energy availability and time.
  • Energy Generation Tracking uses a linked list to store energy generation records and track the total energy produced by different renewable sources.

These solutions are designed to optimize renewable energy use in agriculture and help farmers track energy generation for better planning and resource management.
