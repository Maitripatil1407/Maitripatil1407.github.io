Smart City Project Concepts

1. Hash Tables

Description: Hash tables are used for efficient storage and retrieval of farm data like soil quality, crop type, and yield history.

Example: Mapping farm IDs to their respective data.

hash_table = {
    "Farm1": {"SoilQuality": "High", "CropType": "Wheat"},
    "Farm2": {"SoilQuality": "Medium", "CropType": "Rice"}
}

# Accessing data
print(hash_table["Farm1"]["CropType"])  # Output: Wheat

2. Queues

Description: Queues are used in solar-powered irrigation scheduling to prioritize water distribution tasks.

Example: Scheduling irrigation tasks based on crop water requirements.

from collections import deque

irrigation_queue = deque(["Field1", "Field2", "Field3"])

# Process irrigation tasks
while irrigation_queue:
    field = irrigation_queue.popleft()
    print(f"Irrigating {field}")

3. Graphs and Linked Lists

Description: Graphs and linked lists are used in supply chain tracking to represent and manage farm-to-market routes.

Example: A graph representing transportation routes.

graph = {
    "Farm": ["Market1", "Market2"],
    "Market1": ["Warehouse1"],
    "Market2": ["Warehouse2"]
}

# Display connections
for node, neighbors in graph.items():
    print(f"{node} connects to {', '.join(neighbors)}")

4. Minimum Spanning Tree (MST), DFS, and BFS

Description: Used to optimize microgrid layout for rural farms.

Example: Applying Prim's algorithm for MST calculation.

import heapq

def prims_algorithm(graph, start):
    mst = []
    visited = set()
    edges = [(0, start, None)]

    while edges:
        cost, current, prev = heapq.heappop(edges)
        if current not in visited:
            visited.add(current)
            mst.append((prev, current, cost))

            for neighbor, weight in graph[current]:
                if neighbor not in visited:
                    heapq.heappush(edges, (weight, neighbor, current))

    return mst

5. Arrays and Stacks

Description: Arrays and stacks are used in energy generation tracking and farmer marketplace data handling.

Example: Tracking energy generation.

energy_data = [100, 200, 150, 300]  # Energy generated per day

def track_energy(data):
    for day, energy in enumerate(data, start=1):
        print(f"Day {day}: {energy} kWh")

track_energy(energy_data)
